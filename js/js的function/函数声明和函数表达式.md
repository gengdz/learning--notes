# 函数声明和函数表达式

## 什么是函数声明

```javascript
function sayHi() {
  console.log('hello')
}
```



## 什么是函数表达式

```javascript
const sayHi = function()  {
  console.log('hello')
}
```



## 区别

函数声明 **可以先调用后定义**。原因是变量提升。

```javascript
getName() // oaoafly
function getName() {
  console.log('oaoafly')
}
```

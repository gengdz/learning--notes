# 工具

## quickfix

quickfix 窗口也有其特别之处。如果将光标置于某条列表项 并按 <CR> 键，Vim将会打开相应的文件，并将光标置于包含匹配结果 的那一行上。

| 命令   | 用途    | 示例 |
|--------------- | --------------- | --------------- |
| `:cfdo{cmd}`   | 在 quickfix 列表中的每个文件执行执行{cmd}   | `:cfdo update ` 更新文件; `:cfdo %s//Practical/g \| update` 合并为一条命令|
| Item2.1   | Item2.2   |          |
| Item1.3   | Item2.3   |          |
| Item1.4   | Item2.4   |          |

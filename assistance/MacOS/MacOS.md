# MacOS


## 快捷键
新建文件夹：`cmd + shift + n`
复制粘贴：`cmd + c` —> `cmd + v`
剪切粘贴：`cmd + c` —> `cmd + v + option`

复制文件路径：`cmd + c + option`

屏幕截图的快捷键：`cmd + control + shift + 4`

切换桌面：`ctrl + < >` 、`crtl + 0-9`

删除文件：`cmd + delete` 删除到垃圾桶，再次对这个文件执行会恢复文件

清空垃圾桶：`cmd + delete + shift`
显示或者隐藏“程序坞”：`cmd + option + d`

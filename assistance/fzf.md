# fzf

模糊搜索工具，可以搜文件

## 使用

| 命令  | 作用                                             |
| ----- | ------------------------------------------------ |
| `C-r` | 历史中搜索                                       |
| `C-t` | 搜索当前文件夹中的内容                           |
| `Tab` | 多选                                             |
| `A-c` | 列出当前文件夹下的目录，选择后会直接进入该文件夹 |

可以使用 `vi C-t` 的方式使用 vim 打开

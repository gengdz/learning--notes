# XSS攻击
## 什么是XSS攻击？
XSS是 Cross-Site Scripting(跨站脚本攻击)的简称,为了和CSS区分，所以把攻击的第一个字母改为X

## XSS分类
根据攻击的来源，可以分为存储型、反射型、DOM型。

| 类型      | 存储区                  | 插入点          |
| --------- | ----------------------- | --------------- |
| 存储型XSS | 后端数据库              | HTML            |
| 反射性XSS | URL                     | HTML            |
| DOM型XSS  | 后端数据库/前端存储/URL | 前端/javaScript |

# 泛型
在定义函数、接口、类的时候，不预先指定具体的类型，而在使用的时候再指定的一种特性



## 函数的泛型
### 普通函数的泛型
```typescript
function createArray<T>(length: number, value: T): Array<T> {
    let result: T[] = [];
    for (let i = 0; i < length; i++) {
        result[i] = value;
    }
    return result;
}

createArray<string>(3, 'x'); // ['x', 'x', 'x']
```



### 箭头函数的泛型
```typescript
const student2 = <T>(value: T): T => {
  return value;
};
```

```html
<style>
    h1 {
      font-size: 2.8em;
      color: #d33682;
      margin: 0.75em;
    }
    h2 {
      font-size: 2.4em;
      color: #9B31EA;
      margin: 0.75em;
    }
    h3 {
      font-size: 1.8em;
      color: #338000;
      margin: 0.75em;
    }
    h4 {
      font-size: 1.4em;
      color: #d33682;
      margin: 0.75em;
    }
</style>
```

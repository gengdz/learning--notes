# 场景化学习

## 实现一个 useState

```typescript

const useState = (initValue) => {
  let state = initValue;

  const setState = (newValue) => {
    if(typeof newValue === function) {
      value  = newValue(value)
    }
    else {
      value = newValue;
    }

    // 更新 UI
    updateUI()
  }

  return [state, setState]
}
```

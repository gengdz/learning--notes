# 内置对象

## encodeURI

将特定字符的每个实例替换为一个、两个、三个或四个转义序列来对统一资源标识符 (URI) 进行编码。

以下字符不进行编码。
类型 包含
保留字符 ; , / ? : @ & = + $
非转义的字符 字母 数字 - \_ . ! ~ \* ' ( )
数字符号 #

## encodeURIComponent

将特定字符的每个实例替换成代表字符的 UTF-8 编码的一个、两个、三个或四个转义序列来编码 URI

与 encodeURI() 相比，此函数会编码更多的字符

不转义的字符：
A-Z a-z 0-9 - \_ . ! ~ \* ' ( )

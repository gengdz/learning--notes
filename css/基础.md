## CSS中position属性

| 值       | 描述                                           |
| -------- | ---------------------------------------------- |
| static   | 默认值，没有定位                               |
| absolute | 绝对定位                                       |
| relative | 相对于其正常位置进行定位                       |
| fixed    | 生成绝对定位的元素，相对于浏览器窗口进行定位。 |
| inherit  | 从父元素继承position属性的值                   |

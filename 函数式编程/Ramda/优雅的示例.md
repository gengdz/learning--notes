# Ramda中值得借鉴的示例

## 返回一个函数的示例

```javascript
// 返回的是一个函数
const applyFav = list =>
  R.ifElse(
    R.compose(
      R.flip(R.contains)(list),
      R.prop("id")
    ),
    R.assoc("faved", true),
    R.identity
  );

const applyFavListToProducts = R.mergeWith(
  (list, target) => R.map(applyFav(list), target),
  favList
);

const addFavToProducts = R.evolve({ groupedProducts: applyFavListToProducts });
addFavToProducts(state);
```

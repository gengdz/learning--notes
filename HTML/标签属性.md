# 标签属性

## 全局属性

每个标签都拥有的属性。



### 常见全局属性

| 属性      | 说明                    | 示例                                                  |
| --------- | ----------------------- | ----------------------------------------------------- |
| title     | 鼠标 hover 时显示的内容 |                                                       |
| style     |                         | `<h1 style="color:blue; text-align:center">word</h1>` |
| class     |                         |                                                       |
| id        |                         |                                                       |
| accesskey | 激活元素的快捷键        |                                                       |
|           |                         |                                                       |
|           |                         |                                                       |



### 用法示例

```html
<h1 style="color:blue; text-align:center">This is a header</h1>
<h2 style="color:bulue; text-align:center">
    这是header标签
</h2>
```
